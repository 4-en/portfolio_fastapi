# HELLO_WORLD

Change this content by editing static/markdown/index.md
You can write using [Markdown](https://www.markdownguide.org/basic-syntax/).

index.md content is displayed on the home page ("/" route), before the list of entries.
You can add other markdown files in the static/markdown/ folder and link to them from here.
Routes for these files are automatically created based on the filename (e.g. static/markdown/about.md is available at "/about").
